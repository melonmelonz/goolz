Speedrun Split Logger

The brief said: pick anything that uses an input + a button + DOM update. I run speedrun categories. So: a split logger. Start the timer, hit Split at every checkpoint, see segment times.

① The Idea

Stripped to the essentials of what an actual splits app does:

  • Type a split name (e.g. Castle 1, Bowser).
  • Hit Split. Capture performance.now().
  • Show: split name · total time · segment time (since previous split).
  • Reset wipes everything.

Numbered list output. Same DOM-update pattern as Ex 2 and 3 — just with an array I keep appending to instead of a single sentence.

② The Prompt I Sent

Make me a vanilla-JS speedrun split logger. One text input for the split name, two buttons: "Split" and "Reset". Use performance.now() for timing. On Split, push {name, total_ms, segment_ms} to an array and re-render an <ol>. Reset clears the array and the list. preventDefault on the form submit.

③ Decisions From The First Draft

BitAI's first tryVerdictWhy
Time source Date.now() REJECT System clock can jump (NTP, DST). performance.now() is monotonic — never goes backwards. The right tool for elapsed time.
Render strategy Append a new <li> via createElement ACCEPT Faster than re-rendering the whole list. Safer than string templates. Builds DOM nodes directly. No injection surface.
Format Raw milliseconds: 4321ms MODIFY Speedrunners read m:ss.mmm. Wrote a tiny fmt() that pads zeros.
Empty name Allowed MODIFY Auto-fill Split N if empty so I don't have to type during the run.
Reset list.inner­HTML = '' MODIFY Use list.replaceChildren(). Same effect, no innerHTML reach.

    ⑤ Notes On The Build

    • The clock is a 60Hz requestAnimationFrame loop. It starts on the first split and keeps ticking until reset. Cheap, smooth, matches the LCD aesthetic.
    • Holding state in a splits = [] array (instead of reading the DOM) is the right move. The DOM is for showing; the array is the source of truth.
    • performance.now() returns a high-res float in milliseconds. Subtract two readings → segment time, no clock-skew bugs.
    • If I pushed this further: localStorage to save personal bests, a "gold splits" highlight, keyboard shortcut for split (Space), category select. Out of scope for today.
