▶ EXERCISE 3 / 5

2-Question Survey

Two inputs, one button, one summary. Then stretch it to three — and see how each new field multiplies the work.

① The Prompt

Build a 2-question survey: a text input asking for a name, and a number input asking for years of programming experience. On submit, write a sentence into a results paragraph that uses both answers. preventDefault, addEventListener, no frameworks.

② What GPT Sent Back

<form id="survey">
  <label>Name <input id="q-name" required></label>
  <label>Years coding <input id="q-years" type="number" min="0" required></label>
  <button>Submit</button>
</form>
<p id="result"></p>

<script>
  document.getElementById('survey').addEventListener('submit', (e) => {
    e.preventDefault();
    const name  = document.getElementById('q-name').value;
    const years = document.getElementById('q-years').value;
    document.getElementById('result').textContent =
      `${name} has been writing code for ${years} year(s).`;
  });
</script>
PENN One thing I caught: .value on a number input is still a string. For "1 year" vs "many years" pluralization I'd want parseInt. Otherwise — accepted.

④ Lab Notes

  • type="number" + min="0" means the browser rejects negative values and non-numeric junk on submit. Free validation.
  • Pluralization with a ternary: years === 1 ? 'year' : 'years'. Tiny detail, but "1 years" reads like a robot.
  • The third field is optional. I check unit.value.trim() before deciding whether to append the extra sentence — if it's empty, don't.
  • Three IDs, three getElementById calls. By Ex 4 this starts to feel repetitive. There's an upgrade waiting (form data, destructuring) but the brief said keep it boring.

⑤ Stretch — Adding A Third Question

The brief asked: how much work was the third input?

  • +1 <label> + <input> in HTML
  • +1 const unit = ... in JS
  • +1 conditional in the output string

Linear cost per field, basically. Doesn't scale — at ten inputs you'd want to loop over the form's FormData or build the inputs from an array. Right now, three is the sweet spot for "type it all out and learn the shape."
