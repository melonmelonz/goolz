AI Doubler

Take a number, double it. Sounds trivial. The AI's first answer still had two real bugs. This is the whole point of the course.

① The Prompt

Write me a small HTML page with one number input and a button labeled "Double it". When the user clicks the button, show twice their number on the page. Use addEventListener and preventDefault. Vanilla JS.

② Raw AI Output (Untouched)

<form id="doubler">
  <input id="n" type="number">
  <button>Double it</button>
</form>
<p id="out"></p>

<script>
  const form = document.getElementById('doubler');
  form.addEventListener('click', (e) => {
    e.preventDefault();
    const n = document.getElementById('n').value;
    document.getElementById('out').inner­HTML = n + n;
  });
</script>

③ Decision Table — Accept / Reject / Modify

LineWhat it doesVerdictWhy
<input id="n" type="number"> Number field, no required MODIFY Add required + step="any" so blank submits don't pretend to work and decimals are allowed.
addEventListener('click', ...) Listens for click on the form REJECT Wrong event. Form has a submit semantic — pressing Enter in the input fires submit, not click. Use 'submit'.
const n = ...value Read the input MODIFY .value is a string. n + n would do string concat: "5" + "5" = "55". Cast with Number(n) or parseFloat.
.inner­HTML = n + n Write the result MODIFY No reason to use .inner­HTML for a number — and it'd be unsafe if the input were ever user text. .textContent.
(missing) Validation MODIFY If the cast returns NaN, say so instead of writing "NaN" to the page.
PENN Two real bugs in 8 lines: wrong event, string math. If I'd accepted as-is and just clicked Double-it on 5, the page would confidently say 55. That's the dangerous kind of bug — it looks like it worked.

④ Final Code (After Edits)

<form id="doubler-form">
  <label for="n">Number</label>
  <input id="n" type="number" step="any" required>
  <button type="submit">Double it</button>
</form>
<p id="doubler-out"></p>

<script>
  const form = document.getElementById('doubler-form');
  form.addEventListener('submit', (event) => {
    event.preventDefault();
    const raw = document.getElementById('n').value;
    const n   = Number(raw);
    const out = document.getElementById('doubler-out');
    if (Number.isNaN(n)) { out.textContent = 'not a number :('; return; }
    out.textContent = `${n} × 2 = ${n * 2}`;
  });
</script>
